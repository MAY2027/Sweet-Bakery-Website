🍰Sweet Bakery🍰
架空のベーカリーショップのメニュー一覧ページを作成しました。

## 🛠️ 使用技術
- HTML：ページの構造を作成
- CSS：レイアウトとデザイン
- JavaScript：動的な要素

## 🎯 概要
Sweet Bakeryは、ケーキやスイーツの商品情報を画像付きで一覧表示し、ユーザーに分かりやすく見せるWebサイトです。  
UIを整え、画像とテキストを整列させることで、視覚的に見やすいデザインを目指しました。

## 🤔 工夫した点・学んだこと
- レスポンシブ対応を意識して、画面が崩れないように設計しました。
- HTMLとCSSでのレイアウトの仕組みを深く理解しました。
